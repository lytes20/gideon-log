#### Step 5: Set Up a Dev Server âš¡
At this point, we have a React app, but we need a development server to run it. Since browsers don't understand JSX, we need a bundler to handle it. There a few tools we can use for this such vite, webpack, parcel etc. We'll use Vite because I have come to love this tool ğŸ˜. It's fast, has great documentation, and is easy to use. ğŸš€

***Install Vite:***
```bash
npm install -D vite 
```
***Install the Vite plugin for React:***

```bash
npm install -D @vitejs/plugin-react 
```

***Configure Vite:***
Create a vite.config.js file and add this code:

```jsx
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';

export default defineConfig({
  plugins: [react()],
}); 
```
