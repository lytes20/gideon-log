# Fetching Data in React js 
Let us say you need to fetch and display products in your project

We use fetch API to make the HTTP request to the server.

Endpoint we are using: [dummyjson](https://dummyjson.com/products)

Mental Model here is:


    - You make a request to the server. 
    - Server responds.
    - You get the response, make checks on the response. Store the response data if any a state variable.
    - Now that you have the data in your state, you can display it in your UI.
    - Read from the state variable.

```jsx
import { useEffect, useState } from "react";
import Product from "./Product";

type ProductSummary = {
  id: number;
  title: string;
};

function ProductsList() {
  const [products, setProducts] = useState<ProductSummary[]>([]);
  const [loading, setLoading] = useState(false);

  useEffect(() => {
    const handleFetchProducts = async () => {
      setLoading(true);
      await fetch("https://dummyjson.com/products")
        .then((res) => res.json())
        .then((data) => {
          setProducts(data.products);
          setLoading(false);
        });
    };

    handleFetchProducts();
  }, []);
  return (
    <div>
      <h1>Products</h1>
      {loading ? (
        <p>Loading...</p>
      ) : (
        products.map((product) => (
          <Product key={product.id} title={product.title} />
        ))
      )}
    </div>
  );
}

export default ProductsList;
```
