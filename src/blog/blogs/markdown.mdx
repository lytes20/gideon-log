# What is markdown?
To help you understand what markdown is, we need to first start by talking about one of the problems it solves - Documentation.

What is Documentation? Documentation is a means of passing knowledge from one source to another.
We accumulate knowledge through various experiences.  If the knowledge one accumulate is never documentated, that knowledge would be lost should one cease to exist.
Documentation is a very important thing.

There are many mediums through which one can document what ever they would like to say.
You could decide to record audio of yourself, 
You could decide to record a video. 
You could draw a picture. You could use your smart phone to take a picture. Or you could decide to use text. 
Let us take text. If you choose to use text there are many ways you can use text. 
When you use text, One way of using text is by creating a text document. Your text document could be structured or unstructured. 
Over the years, brilliant smart people have come up with various structured ways you could structure a document.
Some examples of ways you could structure your document include:
- XML
- HTML
- JSON
- markdown.

So what is markdown.
Just think of markdown as a way you can document your text in a structured manner. 
Here is a list of Markdown allows you to add 
- headings, 
- paragraphs,
- Lists, numbered lists, unnumbered lists
- images

Markdown has become very powerful and useful
This very blog you are reading I wrote it using MDX. A blend of  markdown and JSX

Markdown documents are easy to read and parse. They can easily be converted to other document formats like pdf, html documents, 

Many AI agents can easily read and understand markdown documents.
